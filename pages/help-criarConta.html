Como faço para criar uma conta do Facebook?

Observação: você precisa ter pelo menos 13 anos de idade para criar uma conta do Facebook.

Para criar uma conta do Facebook:

  1. Acesse facebook.com e clique em Criar nova conta.
  2. Insira seu nome, senha e foto.
  3. Clique em Cadastrar-se.
  4. Para terminar de criar sua conta, você precisará confirmar suas informações revisando-as

Se estiver tendo problemas para criar uma conta do Facebook:

Para nos avisar sobre o problema que você está enfrentando ao criar uma conta do Facebook, vá para a pagina de help

Se estiver com problemas para entrar:

  • Se você estiver com problemas com a senha, saiba como redefini-la.
  • Caso ainda não esteja conseguindo entrar na sua conta, veja o que pode tentar fazer.
